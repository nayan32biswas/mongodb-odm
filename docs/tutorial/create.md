## Create data with MongoDB console

Let's create the `player` collection in `test_db` and insert one player record using the **MongoDB console**.

```shell
use test_db
db.player.insertOne({
    "name": "Pelé",
    "country_code": "BRA",
    "rating": null,
})
```
